## Tech Stack

- **Language:** Python 3.12
- **Framework:** FastAPI
- **Database:** PostgreSQL
- **Architecture:** Clean Architecture + Domain-Driven Design

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
uvicorn src.presentation.http.main:app --reload

# Run tests
pytest tests/
```
